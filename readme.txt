== Installation ==

1. Download the plugin
2. Unzip mortgage-calculator-sidebar-widget.zip
3. Upload the entire mortgage-calculator-sidebar-widget folder to your plugins directory
4. Go to the Plugins page in your WordPress Administration area and click to 'Activate' mortgage calculator sidebar widget on WordPress.
5. Go to Appearance->widgets and add the mortgage calculator sidebar widget on WordPress widget to your sidebar
